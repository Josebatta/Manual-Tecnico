Justificacion

Este manual fue creado con fines de educar a los usuarios y explicarles de forma específica cómo hacer los dos tipos de mantenimientos del computador y cuáles podrían ser algunos de los materiales caseros que podrían ser usados para estos mismos fines y las instrucciones para que sean realizados de forma correcta, un manual es algo muy importante que esté presente en mano de cualquier estudiante de informática para poder saber diversas cosas como por ejemplo este que ha sido creado para educar sobre el mantenimiento de un computador y los materiales caseros que podrían ser usados para esto mismo, por lo cual tenemos que tener en cuenta que la creación de un manual puede mejorar la capacidad de una persona que haya estudiado o esté estudiando informática para realizar el mantenimiento adecuado de un computador.
